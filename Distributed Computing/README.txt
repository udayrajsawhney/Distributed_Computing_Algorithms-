1> To run project, jar (Distributed Computing.jar) has been created.
2> To view project, open using any IDE such as (Eclipse, IntelliJ).
3> Run by executing the commamd : java -jar Distributed\ Computing.jar
4> Project folder : ( src ) contains to packages : 
	a) alternatealgorithm
	b) sasakialgo
5> Space Complexity : (O(n)) 
6> Time Complexity : (O(n^2))
7> Data Structures Used : 
	a) ArrayList (Dymanic Data Structure Class Implementation of List Interface).
	b) Queue (Queue Data Structure Class Implementation of Linked List Interface).
